JSON Import

Backup auswählen oder JSON einfügen.

Kurzhilfe

  • Speichern: automatisch lokal im Browser; Strg+S speichert sofort.
  • Kalender: ein Tag filtert die Liste; „+ am Tag“ legt dort eine Notiz an.
  • Backup: Export erstellt JSON, TXT erstellt eine lesbare Textsammlung.
  • Bereiche verschieben: Kopfbereich anfassen und Kachel an eine andere Position ziehen.
  • Größe ändern: Kachel unten rechts ziehen. Auf kleinen Bildschirmen bleibt alles untereinander.
  • Barrierearm: Tab-Taste springt durch Bedienelemente; gelber Rahmen zeigt den Fokus.
  • Filter zurücksetzen: „Reset“ zeigt wieder alle Notizen.
  • Profi-Songeditor: arbeitet separat. Bereichsvorlagen per Klick oder Drag & Drop in den Songtext einfügen.
